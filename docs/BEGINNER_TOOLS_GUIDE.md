## 🎯 S级 · 必备编辑器

### 1. Obsidian - 你的第二大脑

**官网**: https://obsidian.md/

**核心特点**:
- **本地优先**: 所有笔记以 Markdown 纯文本存储在本地，完全掌控自己的数据
- **双向链接**: 支持笔记间的双向链接，构建个人知识网络
- **图谱视图**: 可视化笔记之间的关系，洞察知识结构
- **Canvas 白板**: 无限画布，适合头脑风暴和项目规划
- **强大插件**: 丰富的社区插件生态，高度可定制

**适用场景**:
- 个人知识管理 (PKM)
- 项目文档整理
- 学习笔记归档
- 思维导图构建

**推荐理由**: 
作为"第二大脑"，Obsidian 帮助你收集、组织、检索和分享知识。它的本地存储方式确保数据安全，Markdown 格式保证未来兼容性。

---

### 2. VSCode / Trae.cn - AI 原生编辑器

#### VSCode
**下载**: https://code.visualstudio.com/

**核心特点**:
- 免费开源，跨平台支持
- 百万级插件生态
- 强大的调试功能
- 内置 Git 版本控制
- 智能代码补全 (IntelliSense)

**适用场景**: 全栈开发、脚本编写、文本编辑

---

#### Trae.cn (国内首款 AI IDE)
**下载**: https://www.trae.cn/solo

**核心特点**:
- **AI 原生集成**: 深度集成 AI 问答、智能补全、自动编程
- **Builder 模式**: 从需求到项目全自动生成（代码+依赖+调试）
- **多模态支持**: 上传设计稿即可生成对应代码
- **本土化优化**: 专为中文开发者定制，支持中文注释智能识别
- **性能优越**: 启动速度快，内存消耗低

**适用场景**: 
- AI 辅助编程
- 零基础快速上手
- 前端开发（设计稿转代码）
- 追求极致效率的开发者

**推荐理由**: 
Trae 是字节跳动推出的 AI 原生编辑器，相比传统 IDE，它能理解自然语言需求并自动生成代码。特别适合非程序员入门 AI Coding。

## ⚡ A级 · 效率增强

### 3. Get笔记 - 语音录入神器

**官网**: https://www.biji.com/note

**核心特点**:
- **AI 语音转文字**: 录音自动转写，支持方言识别
- **智能文本优化**: AI 自动润色和整理录音内容
- **多设备同步**: 手机、电脑、微信小程序实时同步
- **离线会议录音**: 自动捕获并总结会议要点
- **GetSeed AI 录音卡**: 专用硬件，自动上传同步

**适用场景**:
- 会议记录
- 课程笔记
- 灵感捕捉
- 访谈整理

**推荐理由**: 
用语音代替打字，效率提升 10 倍。AI 自动整理让你专注思考，而非记录。

## 🛠 B级 · 必学技能平台

### 4. GitHub - 代码托管与协作

**注册**: https://github.com/

**核心特点**:
- 全球最大的代码托管平台
- Git 版本控制
- 开源项目学习
- 团队协作工具
- GitHub Actions 自动化部署

**适用场景**:
- 代码版本管理
- 开源项目参与
- 技术简历展示
- CI/CD 自动化

**推荐理由**: 
GitHub 是程序员的"名片"。即使不写代码，也能通过 GitHub 学习优秀项目、管理文档、搭建个人博客。

---

### 5. Google AI Studio - AI 能力试验场

**访问**: https://aistudio.google.com/

**核心特点**:
- 免费体验 Google Gemini 系列模型
- 支持多模态输入（文本、图片、视频）
- Prompt 工程实验
- API 集成测试
- 无需付费即可体验最新 AI 能力

**适用场景**:
- AI 模型能力测试
- Prompt 调优实验
- 多模态应用开发
- 学习 AI API 集成

**推荐理由**: 
Google AI Studio 是免费且强大的 AI 实验平台，特别适合新手了解 AI 模型能力边界和 Prompt 工程。

## 💡 使用建议

### 新手入门路线
1. **第一步**: 注册 GitHub，了解版本控制基础
2. **第二步**: 安装 Obsidian，开始构建个人知识库
3. **第三步**: 下载 Trae.cn 或 VSCode，尝试 AI 辅助编程
4. **第四步**: 体验 Google AI Studio，了解 AI 模型能力
5. **第五步**: 关注宝玉和向阳乔木，持续学习前沿知识

### 工具组合推荐
- **知识管理**: Obsidian + Get笔记 + 博主推荐内容
- **AI 编程**: Trae.cn + GitHub + Google AI Studio
- **效率提升**: Get笔记（语音录入）+ Obsidian（整理归档）
